Crear repositorio local y subir a github
Para crear guardar el repositorio local a GIT HUB. primero se tiene que crear un nuevo repositorio en GITHUB desde el cual se tien que copiar el url del repositorio
  • habrir Git Bash Here desde dentro del derictorio.
    • Click derecho y seleccionar Git Bash Here. (Se abrira laconsola GIT con la direccion del directorio).
  • Para Inicializar el repositorio OBLIGATORIO
    • git init
  • Para verificar que es lo que se va subir (Los archivis que se mandara a repositoroio) --"este paso no es obligatorio"
    • git status
  • Para Agregar al repositorio OBLIGATORIO
    • git add .
  • Para hacer commit OBLIGATORIO
    • git commit -m "Mi primer commit"
  • Luego le pedira el correo OBLIGATORIO
    • git config --global user.email salovaldez.net@gmail.com
    • git config --global user.name SaloValdez
      • no se podia el antes por que primero se tenia que configirar el usuario
    • git config --global user.email salovaldez.net@gmail.com
  • SUBIR A GITBUB-- 1. subir a rama principal OBLIGATORIO
    • git config --global user.email salovaldez.net@gmail.com
  • SUBIR A GITBUB-- 2. subir a repositorio GITHUB (Copiar y pegar--copiar el url del repositorio github y pegar a la consola Git) OBLIGATORIO
    • git remote add origin https://github.com/SaloValdez/Ejemplo_repo.git
  • SUBIR A GITBUB-- 3. hacer push para mandar al repositorio GITHUB (Copiar y pegar--copiar el url del repositorio github y pegar a la consola Git) {Posiblemente pida logearse a travez de consola o navegador } OBLIGATORIO
    • git push -u origin main
